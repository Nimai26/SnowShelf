## Technologies

### Backend
- **NestJS** 10.x - Framework Node.js avec TypeScript
- **TypeORM** 0.3.x - ORM pour MariaDB
- **MariaDB** 10.11 - Base de données relationnelle
- **Redis** 7 - Cache et sessions
- **JWT** - Authentification stateless
- **Swagger** - Documentation API interactive

### Frontend
- **React** 18.x - Bibliothèque UI
- **TypeScript** 5.x - Typage statique
- **Vite** 5.x - Build tool ultra-rapide
- **TailwindCSS** 3.x - Framework CSS utility-first
- **React Router** 6.x - Routing côté client
- **TanStack Query** 5.x - Gestion des données serveur
- **Zustand** 4.x - State management léger
- **Axios** - Client HTTP

### APIs externes
- **Tako API** - API unifiée de recherche (32 providers, 11 domaines)
- **FlareSolverr** - Contournement de protections anti-bot

### DevOps
- **Docker** & **Docker Compose** - Conteneurisation
- **GitHub Actions** - CI/CD
- **NGINX Proxy Manager** - Reverse proxy et certificats SSL
- **phpMyAdmin** - Administration de base de données
- **MailHog** - Email testing en développement

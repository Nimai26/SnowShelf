# Images du projet SnowShelf

Ce dossier contient les images statiques du site.

## Structure

```
images/
├── backgrounds/     # Fonds d'écran (login, etc.)
│   └── login-bg.jpg # Fond de la page de connexion
├── logo.png         # Logo SnowShelf
├── mascot.png       # Mascotte (bonhomme de neige)
└── favicon.png      # Favicon du site
```

## Notes

- Les images de fond doivent être optimisées pour le web (max 500KB)
- Formats recommandés : WebP, JPEG, PNG
- Dimensions recommandées pour le fond de login : 1920x1080 minimum
